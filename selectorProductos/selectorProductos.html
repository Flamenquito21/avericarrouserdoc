Pantalla de carga Inicio Menú

Selector de Productos

Este menú estará en todas las vistas a partir de ahora. Cuenta con varias opcines.

Listas favoritas

Una vez pulsado el botón de “Nueva Lista”, se accede a la vista en la cual se seleccionan los productos que van a formar la lista. Esta vista está formada inicialmente por los siguientes apartados.

el buscador

En el buscador se escribe el nombre del producto el cual se quiere añadir a la lista. Una vez escrito, se desplegará el producto con el nombre y se pulsará el botón verde con el icono del carro “añadir a cesta”.

Botón de unidades

Junto con el buscador, hay un botón “Unidades” azul, el cual dictará cuantas unidades se van a añadir de ese producto.

Mi cesta

Botón que al pulsar accederá a la lista actual, la lista a la que se añaden los productos. Como podemos observar, la cesta tiene varios componentes:

Productos

Los productos añadidos desde el buscador. Para eliminar un producto solo tendremos que deslizarlo hacia la izquierda y aparecerá una papelera blanca sobre un fondo rojo.

Guardar lista

Tras pulsar este botón verde que se encuentra en la parte inferior izquierda de la pantalla, automáticamente se guarda la lista actual con el nombre que escojamos.

Confirmar lista

Al seleccionar este botón verde el cual se encuentra en la parte inferior derecha de la pantalla, confirmaremos que la lista está finalizada y se aplicará el logaritmo que decidirá la mejor opción de compra. Una vez pulsado el botón aparecerá una nueva vista con el resultado del logaritmo.

Producto favoritos

Pulsando la estrella que se encuentra al lado de cada producto de la lista, se añadirá dicho producto a la lista de productos favoritos. Para eliminar algún producto, deslizamos dicho producto hacia la izquierda y nos aparecerá un icono de una papelera blanca sobre un fondo rojo.
